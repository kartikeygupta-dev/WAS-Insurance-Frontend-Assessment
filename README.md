This project is part of the frontend developer assessment for WAS Insurance (Wise and Silent).

The goal of this task was to translate high-fidelity Figma designs into clean, responsive, and high-performance web pages using strong frontend fundamentals.

---

## Pages Implemented

1. Home Page (Desktop)
2. Home Page with Desktop Mega Menu
3. Home Page with Mobile Navigation Menu
4. Hero Section
All pages are built based on the provided Figma design.

---

## Tech Stack

- HTML5 (semantic markup)
- CSS3 (responsive layout, clean styling)
- JavaScript (menu interactions)
- Mobile First Approach

No frontend frameworks such as React, Angular, or Vue were used.

---

## Features

- Pixel-accurate Figma to HTML conversion
- Fully responsive across desktop, tablet, and mobile devices
- SVG logo for sharp rendering and better performance
- Desktop mega menu interaction
- Mobile-friendly navigation menu
- Optimized for performance, accessibility, and SEO
- Cross-browser compatible

---

## Performance & Optimization

- Optimized images and SVG assets
- Proper heading hierarchy and semantic HTML
- Accessible navigation and buttons
- Meta tags for SEO
- Tested using Google PageSpeed Insights
